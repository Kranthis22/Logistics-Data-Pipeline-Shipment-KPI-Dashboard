# Data Pipeline Architecture

This project simulates a logistics data pipeline inspired by real-world data engineering workflows.

## Flow:

1. Source Data (CSV files)
2. Data Ingestion (Azure Data Factory - Concept)
3. Data Processing (Python)
4. Data Transformation (SQL)
5. Data Storage (Warehouse concept)
6. Reporting Layer (Power BI)

This represents a simplified version of enterprise data pipelines used in logistics and operations.
