# Logistics Data Pipeline & Shipment KPI Dashboard

This project is inspired by my experience working on data engineering and reporting solutions.

I built a simple pipeline to process shipment data, perform transformations using SQL and Python, and support KPI reporting using Power BI.

## Project Structure
- data
- sql
- python
- docs

## KPIs
- Total Shipments
- Average Delivery Time
- Late Deliveries
## Data Flow

- Source data is loaded from CSV files
- Data is cleaned and processed using Python
- SQL is used to calculate key metrics
- Final data is used for reporting in Power BI

## Tools Used

- Azure Data Factory (conceptual pipeline)
- SQL
- Python (Pandas)
- Power BI

## Use Case

This project is based on a logistics operations scenario where shipment data is processed to monitor delivery performance and operational KPIs.
